# The Art of War Skills

孫子兵法，是春秋末期的齊國人孫武撰寫的兵書，是中國最高的戰爭哲學，也是作戰原理的經典。不僅用於戰爭情勢的分析，更用於決策分析；
蒸餾孫武的思想與戰爭哲學，讓孫武成為你的個人必勝軍師。

---


## Skills

| Skill | Description |
| --- | --- |
| `strategic-planning` | **【計】** 評估自身優劣勢與外部環境，在行動前進行勝率推算與長遠佈局。 |
| `cost-efficiency` | **【作戰】** 管理時間、精力和金錢成本，講求高效率、速戰速決，避免陷入內耗。 |
| `win-win-negotiation` | **【謀攻】** 以智慧、談判或借力打力來達成目標，追求傷害最小、利益最大的雙贏。 |
| `risk-resilience` | **【形】** 先做好自我防護（如存好預備金、充實底氣），讓自己立於不敗再尋找機會。 |
| `momentum-building` | **【勢】** 累積常態實力（正），並在關鍵時刻運用爆發力（奇），創造勢不可擋的成果。 |
| `leverage-points` | **【虛實】** 避開競爭最激烈、對手最強的地方，集中資源投入在最容易突破的關鍵點。 |
| `first-mover-advantage` | **【軍爭】** 在機會出現時快速反應，比別人搶先一步佔領有利位置，化被動為主動。 |
| `agile-adaptation` | **【九變】** 不死守教條，規矩是死的、人是活的，能根據當前現實狀況靈活轉彎。 |
| `trend-spotting` | **【行軍】** 敏銳觀察生活或市場周遭的蛛絲馬跡，在危機或機會來臨前就提早預判。 |
| `boundary-awareness` | **【地形】** 摸清不同環境的隱形規則與邊界限制（如公司文化、法律防線），避免踩雷。 |
| `mindset-management` | **【九地】** 根據自己或團隊當下所處的處境（如順境、絕境），調整心理狀態與行動手段。 |
| `leveraged-catalyst` | **【火攻】** 善用外部重大推力（如新科技、爆紅話題、社群槓桿）來加速局勢發展、打破僵局。 |
| `networking-intelligence` | **【用間】** 廣結善緣、建立多元資訊管道，靠精準的情報做出正確決策，而不是瞎猜。 |
